# Security policy

## Supported versions

Security fixes are applied to the latest release and `main`.

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security reports.

Email **hoffmang@hoffmang.com** with:

- A description of the issue and potential impact
- Steps to reproduce, if applicable
- Your macOS version and Porthole build (release tag or commit)

We aim to acknowledge reports within a few days and will coordinate disclosure
once a fix is available.
